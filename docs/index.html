Query syntax and operators

search using NOT AND For inclusive ranges use square brackets: [min TO max]. For exclusive ranges use curly brackets: {min TO max}. Curly and square brackets can be combined in the same range expression: logins_count:[100 TO 200}. Use ranges in combination with wildcards. For example, to find all users with more than 100 logins, use q=logins_count:{100 TO *].
